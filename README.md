# Email Marketing Dashboard | Power BI

## Project Summary
Developed an Email Marketing Analytics dashboard using Power BI to analyze customer
sign-ups, user engagement, and campaign performance. The dashboard enables marketing
teams to make data-driven decisions to improve conversion and engagement.

## Key Responsibilities
- Cleaned and prepared marketing data for analysis
- Created DAX measures for KPIs and metrics
- Built interactive dashboards using Power BI
- Analyzed user activity, demographics, and time-based trends

## Tools & Technologies
- Power BI Desktop
- DAX
- Data Cleaning
- Data Modeling
- Data Visualization
- Marketing Analytics

## Key Metrics
- Total People Signed Up
- Active vs Inactive Users
- Sign-ups by State and City
- Sign-ups by Time and Year
- Demographic-based Enrollment

## Insights & Outcomes
- Identified peak sign-up time periods
- Analyzed regional performance across states and cities
- Discovered higher engagement among married individuals
- Highlighted trends to optimize future email campaigns
